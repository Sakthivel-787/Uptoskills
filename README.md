
Project Title: AI Based Talking Image System

Description:
This project converts a static image into a talking video.
The image speaks any content provided in the text file.

Technologies Used:
- Python
- VS Code
- Text to Speech (gTTS)
- Wav2Lip Deep Learning Model

Usage:
1. Put image in input/image.jpg
2. Write content in input/text.txt
3. Run python generate_video.py
4. Output video will be in output/result.mp4
